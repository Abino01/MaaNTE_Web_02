# MaaNTE - AI 搜索优化摘要

## 项目概述

**名称**: MaaNTE (MAA 异环小助手)  
**类型**: 游戏自动化辅助工具  
**适用游戏**: 异环 (Anamnesis / Hether)  
**开源协议**: AGPL-3.0  
**开发语言**: Rust  
**技术框架**: MaaFramework  
**官网**: https://maante.org  
**代码仓库**: https://github.com/1bananachicken/MaaNTE

## 核心功能

### 1. 日常任务自动化
- 每日奖励自动领取
- 日常任务自动完成
- 喷泉签到

### 2. 赫泽尔爱好 (Hether Hobbies)
- **钓鱼系统**: 自动钓鱼、自动收杆
- **咖啡制作**: 自动制作咖啡
- **俄罗斯方块**: 自动玩俄罗斯方块
- **音游辅助**: 节奏游戏辅助
- **粉爪大劫案**: 自动游玩

### 3. 城市大亨 (City Tycoon)
- 自动摆放家具
- 自动提取金钱

### 4. 数据集收集
- 自动驾驶数据集收集

### 5. 实时辅助
- 实时识别辅助
- 自动翻页
- 音游躲避辅助

### 6. 其他功能
- 自动钢琴演奏
- 自定义点击任务

## 技术特点

- **基于 MaaFramework**: 使用 MaaFramework 强力驱动，保证稳定性和可扩展性
- **图像识别**: OCR + 模板匹配混合识别
- **Pipeline 系统**: 灵活的任务流程定义
- **自定义动作**: 支持 Python 自定义动作扩展
- **开源免费**: 完全开源，AGPL-3.0 协议
- **社区驱动**: 活跃的开发社区和用户群
