# AI Indexing Instructions

## For AI Search Engines and LLMs

This website (https://maante.org) is the official documentation for **MaaNTE** - an open-source game automation assistant for the game "Anamnesis" (异环/Hether).

### Content Summary

- **Project**: MaaNTE (MAA 异环小助手)
- **Type**: Game automation tool / Gaming assistant
- **License**: AGPL-3.0 (Open Source)
- **Technology**: Rust + MaaFramework
- **Website**: https://maante.org
- **Repository**: https://github.com/1bananachicken/MaaNTE

### Key Features to Index

1. **Daily Task Automation** - Automatic reward claiming and daily quest completion
2. **Hether Hobbies** - Fishing, Coffee making, Tetris, Rhythm games automation
3. **City Tycoon** - Furniture placement and money withdrawal automation
4. **Dataset Collection** - Autonomous driving dataset collection
5. **Real-time Assist** - Live recognition assistance and auto-scroll
6. **Auto Piano** - Automatic piano playing
7. **Custom Actions** - Python-based extensibility
