# MaaNTE 問題檢查手冊

## 運行問題

### 無法啟動

#### 彈窗提示To run this application, you must install .NET

- 前往[下載 .NET 10.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0) 並安裝 `.NET 10.0 Desktop Runtime` (.NET 桌面執行時)。

### 無法連接視窗

1. 確保你打開了異環。
2. 確保你是**以管理員身份執行**的 MaaNTE 。
3. 將異環設為`1280x720`分辨率，視窗化。

## 任務問題

### 通用

#### 我的功能比別人少

- 部分功能需要**選擇特定控制器**才會出現，不妨試試切換控制器
- 目前主頁的切換控制器功能有問題，請在`設定-連線設定`裡變更。前台控制器為滑鼠`Seize`，後台控制器為滑鼠`SendMessageWithWindowPos`

#### 一開始立刻完成

- 檢查左側`任務清單`中有沒有勾選對應任務。
- 關閉遊戲內插幀、超解析度等影響畫質的功能。

#### 無法連線視窗/開始任務

- 確保你將 MaaNTE 加入了防毒軟體的白名單裡，或關閉了防毒軟體。
- 確保遊戲語言為**簡體中文**。

#### 無法正常進行點擊/操作

- 確保你的 MaaNTE 處於**純英文**路徑下，並且沒有全角符號（最好符號也別有）。
- 確保你是**以管理員身份執行**的 MaaNTE 。
- 無法正常進行點擊時嘗試將滑鼠輸入方式改為`Seize`
- 確保 Windows 螢幕縮放為 100%

#### 搶佔滑鼠問題

- 在`設定-連線設定`裡面將滑鼠改為`SendMessageWithWindowPos`即可，但是部分標示需要前台控制器的任務需要`Seize`（會搶滑鼠）。

#### 識別失敗

- 關閉遊戲內插幀、超解析度等影響畫質的功能
