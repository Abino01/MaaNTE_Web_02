## 簡介

自動刷取粉爪大劫案。

這個任務需要在小吱身邊開始（即保證螢幕上存在小吱的互動按鈕）。

::: warning

執行本任務前，請務必在 MaaNTE 的設定中設定好`結束任務快速鍵`並開啟`全域快速鍵`。

:::

::: tip

執行本任務前，請在遊戲的設定中關閉`移動鏡頭修正`功能。

:::

::: info 控制器

- [ ] 桌面端-默認
- [x] 桌面端-前台
- [ ] 桌面端-後台

:::

::: steps

1. 無限循環

   啟用後任務將無限循環，直到手動停止。啟用時會隱藏忽略 `循環次數` 設定。

   預設開啟，我們也推薦你開啟該功能。

2. 循環次數

    本次執行要進行多少次該任務。

    填入的內容需要滿足**任意正數** 。

3. 地圖方案

    選擇合適的方案。

    不同方案一般對配隊的要求不同，請依描述配置好編隊。 MaaNTE 並不會更改你的配隊。

   ::: info 注意
   不同方案可能由不同作者編寫，具體的配置項也不同。
   :::
