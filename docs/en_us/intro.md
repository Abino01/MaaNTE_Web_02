::: steps

1. Download

    Obtain through one of the following methods MaaNTE ：

    - [GitHub Releases](https://github.com/1bananachicken/MaaNTE/releases)
    - [Mirror Chan](https://mirrorchan.com/zh/projects?rid=MaaNTE)
    - [Download at QQ group](/en_us/qq-group/)
    - Download at colud：[Baidu colud](https://pan.baidu.com/s/11QMC-aYfjfq52yco_UAwfg?pwd=tkmu) [Quark colud](https://pan.quark.cn/s/4b70d06b913c?pwd=irqh)

    ::: warning
    Recently, it has been discovered that a large number of accounts are redistributing the software, and some accounts are even impersonating official accounts to **spread the virus**.Software obtained through unofficial channels may **contain viruses** and is usually not the latest version. Please always download from the official channels mentioned above.
    :::

2. Install

    Install MaaNTE in a **pure English directory**.

3. Check

    Ensure the software is intact and add it to your antivirus software's whitelist.

    ::: info
    It is currently known that `Windows Defender` (`Windows Security Center`) may cause `Python.exe` to become unusable.
    :::

4. Start the software

    Run `MaaNTE.exe` as an administrator.

5. Configuration

    After opening MaaNTE, you will need to perform some configuration steps.

    - Select controller and connect window

    - Select the task you need to perform.
        ::: info
        You can go to [Feature Introduction](introduction/README.md) to view the task introduction.
        :::

6. Start the mission

    Start the task and enjoy automation.
:::
