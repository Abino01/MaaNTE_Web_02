:::

::: steps

1. 下載

    在下列途徑之一獲取 MaaNTE ：

    - [GitHub Releases](https://github.com/1bananachicken/MaaNTE/releases)
    - [Mirror 酱](https://mirrorchan.com/zh/projects?rid=MaaNTE)
    - [QQ 群下载](/zh_tw/qq-group/)
    - 网盘下载：[百度网盘](https://pan.baidu.com/s/11QMC-aYfjfq52yco_UAwfg?pwd=tkmu) [夸克网盘](https://pan.quark.cn/s/4b70d06b913c?pwd=irqh)

    ::: warning
    近期出現大量帳號二次分送本軟體，甚至冒充官方帳號**傳播病毒**。透過非官方途徑取得的軟體可能**含有病毒**，且一般不是最新版本。請務必從上述官方管道下載。
    :::

2. 安装

    将 MaaNTE 安装在**純英文目錄**中。

3. 檢查

    確保軟體完整，並將其添加到防毒軟體的白名單中。

    ::: info
    目前已知`Windows Defender`（`Windows 安全中心`）可能導致`Python.exe`不可用
    :::

4. 啟動軟體

    以管理員身份執行`MaaNTE.exe`。

5. 配置

    在打開 MaaNTE 後，你需要進行一些設定工作。

    - 選擇控制器並連接視窗

    - 選取你需要進行的任務
        ::: info
        你可以前往[功能介紹](introduction/README.md)查看任務介紹。
        :::

6. 開始任務

    開始任務，享受自動化。
:::
