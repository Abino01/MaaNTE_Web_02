::: steps

1. 下载

    在下列途径之一获取 MaaNTE ：

    - [GitHub Releases](https://github.com/1bananachicken/MaaNTE/releases)
    - [Mirror 酱](https://mirrorchan.com/zh/projects?rid=MaaNTE)
    - [QQ 群下载](/zh_cn/qq-group/)
    - 网盘下载：[百度网盘](https://pan.baidu.com/s/11QMC-aYfjfq52yco_UAwfg?pwd=tkmu) [夸克网盘](https://pan.quark.cn/s/4b70d06b913c?pwd=irqh)

    ::: warning
    近期出现大量账号二次分发本软件，甚至冒充官方账号**传播病毒**。通过非官方途径获取的软件可能**含有病毒**，并且一般不是最新版本。请务必从上述官方渠道下载。
    :::

2. 安装

    将 MaaNTE 安装在**纯英文目录**中。

3. 检查

    确保软件完整，并将其添加到杀毒软件的白名单中。

    ::: info
    目前已知`Windows Defender`（`Windows 安全中心`）可能导致`Python.exe`不可用
    :::

4. 启动软件

    以管理员身份运行`MaaNTE.exe`。

5. 配置

    在打开 MaaNTE 后，你需要进行一些配置工作。

    - 选择控制器并连接窗口

    - 选中你需要进行的任务
        ::: info
        你可以前往[功能介绍](introduction/README.md)查看任务介绍。
        :::

6. 开始任务

    开始任务，享受自动化。
:::
