---

## 簡介

自動完成音遊「超強音」。需要在該玩法介面開始任務。

目前《迷星叫》曲目已經可以達到96%準度

::: info 控制器

- [x] 桌面端-默認
- [x] 桌面端-前台
- [x] 桌面端-後台

:::

::: steps

1. 自動選曲

 開啟時自動選擇演奏《迷星叫》，關閉後可自選`目標歌曲`。

2. 目標歌曲

 自選曲目。

3. 自動連打

 啟用後連續演奏指定次數，關閉則僅演奏一次。

4. 連打直到活力耗盡

 啟用後連續演奏直到沒有足夠都市活力用於領取方斯獎勵。關閉後可設定`連打次數`。

5. 連打次數

 循環演奏多少次。

6. 目標幀率

 演奏循環的每秒幀率，建議`30`-`120`。
:::
