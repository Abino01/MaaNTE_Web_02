title: 超强音
---

## 简介

自动完成音游“超强音”。需要在该玩法界面开始任务。

目前《迷星叫》曲目已经可以达到96%准度

::: info 控制器

- [x] 桌面端-默认
- [x] 桌面端-前台
- [x] 桌面端-后台

:::

::: steps

1. 自动选曲

    开启时自动选择演奏《迷星叫》，关闭后可自选`目标歌曲`。

2. 目标歌曲

    自选曲目。

3. 自动连打

    启用后连续演奏指定次数，关闭则仅演奏一次。

4. 连打直到活力耗尽

    启用后连续演奏直到没有足够都市活力用于领取方斯奖励。关闭后可设置`连打次数`。

5. 连打次数

    循环演奏多少次。

6. 目标帧率

    演奏循环的每秒帧率，建议`30`-`120`。
:::
