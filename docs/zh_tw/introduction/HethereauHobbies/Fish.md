## 简介

自动执行钓鱼任务的挂机功能。包含[旧版](#task1)和[新版](#task2)个任务入口。

## 钓鱼任务{#task1}

自动循环执行钓鱼：抛竿、等待上钩、收杆，支持自动卖鱼和自动买鱼饵。

::: info 控制器

- [x] 桌面端-默认
- [x] 桌面端-前台
- [x] 桌面端-后台

:::

::: steps

1. 无限循环

    启用后钓鱼任务将无限循环，直到手动停止。启用时会隐藏忽略 `循环次数` 设置。

2. 循环次数

    本次执行要进行多少轮次该任务。

    填入的内容需要满足 `^\d+$` ，即 **任意数字** 。

3. 每次钓鱼数量

    每个循环钓多少次鱼，建议数值为`99`。

    ::: info 举个例子
    当`循环次数`为`5`，`每次钓鱼数量`为`99`时，会在钓鱼`99*5=495`次后停止。
    :::

4. 自动卖鱼

    是否在执行一个轮次的钓鱼后进行卖出鱼篓内**全部**鱼的操作。

5. 自动买鱼饵

    是否在执行一个轮次的钓鱼后购买`99`个鱼饵。

    - 鱼饵识别阈值

        购买鱼饵时对识别鱼饵图像的置信度要求，越高越严格。

        无法识别点击鱼饵位置时可以尝试调低该数值。
:::

## 钓鱼任务（新）{#task2}

自动循环执行钓鱼，默认情况下会一直循环执行。

> [!WARNING]
> 仍无法直接处理被月卡打断的跨夜钓鱼，可以尝试设置定时任务来继续钓鱼，实际效果无法保证。
> 钓鱼功能仅自动买鱼饵和自动卖鱼会抢占鼠标。

::: info 控制器

- [+] 桌面端-默认
- [x] 桌面端-前台
- [+] 桌面端-后台

:::

::: steps

1. 自动卖鱼

    是否在**鱼鳞币不足时**自动出售背包中的鱼。

2. 自动买鱼饵

    是否在识别到**鱼饵不足时**购买`99`个鱼饵。

    - 鱼饵识别阈值

        购买鱼饵时对识别鱼饵图像的置信度要求，越高越严格。

        无法识别点击鱼饵位置时可以尝试调低该数值。
:::
