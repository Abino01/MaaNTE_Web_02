## 简介

自动刷取粉爪大劫案。

该任务需要在小吱身边开始（即保证屏幕上存在小吱的交互按钮）。

::: warning

执行本任务前，务必在 MaaNTE 的设置中设置好`结束任务快捷键`并开启`全局快捷键`。

:::

::: tip

执行本任务前，请在游戏的设置中关闭`移动镜头修正`功能。

:::

::: info 控制器

- [ ] 桌面端-默认
- [x] 桌面端-前台
- [ ] 桌面端-后台

:::

::: steps

1. 无限循环

    启用后任务将无限循环，直到手动停止。启用时会隐藏忽略 `循环次数` 设置。

    默认情况下开启，我们也推荐你开启该功能。

2. 循环次数

    本次执行要进行多少次该任务。

    填入的内容需要满足**任意正数** 。

3. 地图方案

    选择合适的方案。

    不同方案一般对配队的要求不同，请按照描述配置好编队。MaaNTE 并不会更改你的配队。

    ::: info 注意
    不同方案可能由不同作者编写，具体的配置项也不同。
    :::
