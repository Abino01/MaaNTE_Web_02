---
title: 实时辅助
---

## 简介

辅助类功能，会**一直运行**直到手动终止。

需要使用前台控制器。!!谁会在后台的时候用到这个东西啊！!!{.mask .hover}

::: warning

执行本任务前，务必在 MaaNTE 的设置中设置好`结束任务快捷键`并开启`全局快捷键`。

:::

::: info 控制器

- [ ] 桌面端-默认
- [x] 桌面端-前台
- [ ] 桌面端-后台

:::

::: steps

1. 检测间隔

    每轮检测之间等待的时间，单位为毫秒。一般情况下无需改动。

2. 自动拾取

    检测到可拾取的东西时按F拾取。!!俺拾嘞!!{.mask .hover}

    ::: warning

    识别的是小手图标，所以说有些虽然不是地上的东西但是用了这个图标的交互也会被按F。

    !!管你这那的，先拿了再说!!{.mask .hover}

    :::

3. 自动传送

    处于地图界面时 ，会在你**选中传送点**时自动点击“传送”按钮。

    目前支持下列传送点：

    - “维特海默塔”
    - “ReroRero电话亭”

4. 自动跳过剧情

    当处于可以跳过的剧情时，自动点击跳过按钮。

    - 自动勾选“今日不再提示”

        在跳过剧情弹窗中，自动勾选“今日不再提示”，避免每次弹出确认

    - 自动跳过剧情概述

        在弹出剧情概述对话框时自动点击“确认”跳过

    - 自动跳过重要剧情提示

        在弹出重要剧情提示对话框时自动点击“确认”跳过
:::
