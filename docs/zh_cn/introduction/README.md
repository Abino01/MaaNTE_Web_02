# 功能介绍

MaaNTE 是面向《异环（Neverness to Everness）》的自动化工具集，专注于减少重复操作、提升日常游玩效率。

## 功能列表

| 功能 | 说明 | 需要前台 |
|------|------|----------|
| [自动钓鱼](HethereauHobbies/Fish.md) | 自动循环钓鱼，支持自动卖鱼与购买鱼饵 | 否 |
| [音频闪避与反击](SoundDodge.md) | 通过音频信号实时执行闪避与反击 | 是（PC 端需保持前台） |
| [超强音](Rhythm.md) | 自动演奏音游，支持选曲与连打 | 否 |
| [实时辅助](RealTimeAssist/RealTime.md) | 自动剧情跳过、自动传送等实时辅助操作 | 是（PC 端需保持前台） |
| [自动做咖啡](MakeCoffee.md) | 自动循环完成咖啡制作小游戏 | 否 |
| [喷泉打卡](FountainCheckin.md) | 自动前往喷泉并完成每日打卡 | 是（PC 端需保持前台） |
| [粉爪大劫案](PinkPawHeist.md) | 自动执行粉爪大劫案活动流程 | 是（PC 端需保持前台） |
| [领取奖励](ClaimRewards.md) | 自动领取可用奖励 | 是（PC 端需保持前台） |
| [领取一咖舍收益](WithdrawMoney.md) | 自动领取一咖舍收益，并可选自动补货 | 是（PC 端需保持前台） |
| [自动弹琴](AutoPiano.md) | 自动演奏钢琴 | 否 |
| [粉爪快速拾取](AutoFScroll.md) | 持续按住 F 并滚动滚轮，实现快速拾取 | 否 |

## 快速开始

1. 在 [Releases](https://github.com/1bananachicken/MaaNTE/releases) 下载最新版本
2. 解压后运行程序
3. 在任务列表中勾选所需功能
4. 点击开始即可运行自动化流程

详细安装与配置说明请查看上游仓库的 [README](https://github.com/1bananachicken/MaaNTE)。

遇到问题？请参考 [问题排查](/zh_cn/问题排查.html) 和 [常见问题](/zh_cn/develop/faq.html)。
