# 功能介绍

MaaNTE 提供了丰富的自动化功能，帮助你简化《异环》中的重复性操作。

## 功能列表

| 功能 | 说明 | 需要前台 |
|------|------|----------|
| [自动钓鱼](Fish.md) | 自动循环钓鱼，支持自动卖鱼和买鱼饵 | 否 |
| [音频闪避与反击](SoundDodge.md) | 基于音频识别的实时闪避与反击 | 是（桌面端前台） |
| [超强音](Rhythm.md) | 自动演奏音游，支持选曲和连打 | 否 |
| [实时辅助](RealTime.md) | 自动剧情跳过、自动传送 | 否 |
| [自动做咖啡](MakeCoffee.md) | 循环执行咖啡制作小游戏 | 否 |
| [喷泉打卡](FountainCheckin.md) | 自动前往喷泉完成每日打卡 | 否 |
| [粉爪劫案](PinkPawHeist.md) | 自动完成粉爪劫案活动 | 否 |
| [领取奖励](ClaimRewards.md) | 自动领取各类奖励 | 否 |
| [取钱](WithdrawMoney.md) | 自动取款操作 | 否 |
| [自动弹琴](AutoPiano.md) | 自动演奏钢琴 | 否 |
| [自动滚动](AutoFScroll.md) | 自动滚动页面 | 否 |

## 快速开始

1. 从 [Releases](https://github.com/1bananachicken/MaaNTE/releases) 下载最新版本
2. 解压后运行可执行文件
3. 在任务列表中选择需要的功能
4. 点击开始，享受自动化！

详细安装和配置说明请查看上游仓库的 [README](https://github.com/1bananachicken/MaaNTE)。

遇到问题？请参考 [问题排查](/zh_cn/问题排查.html) 和 [常见问题](/zh_cn/develop/faq.html)。
