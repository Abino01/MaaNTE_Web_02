# 機能紹介

MaaNTE は『異環（Neverness to Everness）』の反復操作を効率化する、多機能な自動化ツールです。

## 機能一覧

| 機能 | 説明 | 前面表示が必要 |
|------|------|-----------------|
| 自動釣り | 釣りを自動周回し、魚の売却や餌の購入まで対応 | 不要 |
| 音声回避・反撃 | 音声情報をもとにリアルタイムで回避・反撃を実行 | 必要（PC版を前面表示） |
| リズムアシスト | 音ゲー「超強音」の自動演奏、選曲・連打に対応 | 不要 |
| リアルタイムアシスト | ストーリーの自動スキップと自動テレポートを実行 | 必要（PC版を前面表示） |
| 自動コーヒー | コーヒー作成ミニゲームを自動で繰り返し実行 | 不要 |
| 噴水チェックイン | 毎日の噴水チェックインを自動で完了 | 必要（PC版を前面表示） |
| ピンククロー大強奪 | ピンククロー大強奪イベントを自動進行 | 必要（PC版を前面表示） |
| 報酬受け取り | 受け取れる各種報酬を自動で回収 | 必要（PC版を前面表示） |
| 一咖舍収益受け取り | 一咖舍の収益を受け取り、必要に応じて補充を実行 | 必要（PC版を前面表示） |
| 自動ピアノ | ピアノ演奏を自動化 | 不要 |
| 高速拾取 | F キー長押しとホイール操作で高速にアイテムを拾取 | 不要 |

## クイックスタート

1. [Releases](https://github.com/1bananachicken/MaaNTE/releases) から最新版をダウンロード
2. 解凍して実行ファイルを起動
3. タスクリストから必要な機能を選択
4. 開始して自動化をお楽しみください！

インストールや設定の詳細は、上流リポジトリの [README](https://github.com/1bananachicken/MaaNTE) をご確認ください。

問題が発生した場合は、[トラブルシューティング](/ja_jp/トラブルシューティングマニュアル.html) と [FAQ](/ja_jp/develop/faq.html) をご確認ください。
