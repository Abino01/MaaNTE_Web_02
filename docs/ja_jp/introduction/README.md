# 機能紹介

MaaNTE は『異環』の繰り返し操作を自動化する豊富な機能を提供します。

## 機能一覧

| 機能 | 説明 | フォアグラウンド |
|------|------|-----------------|
| 自動釣り | 自動釣り、魚の売却・餌の購入に対応 | 不要 |
| オーディオ回避・反撃 | 音声認識によるリアルタイム回避・反撃 | 必要（デスクトップ） |
| リズムゲーム | 音ゲー「超強音」の自動演奏、選曲・連打対応 | 不要 |
| リアルタイムアシスト | ストーリー自動スキップ、自動テレポート | 不要 |
| 自動コーヒー | コーヒー作りのミニゲームを自動周回 | 不要 |
| 噴水チェックイン | 毎日の噴水チェックインを自動化 | 不要 |
| ピンククロー強奪 | ピンククロー強奪イベントを自動クリア | 不要 |
| 報酬受け取り | 各種報酬を自動で受け取り | 不要 |
| 自動ピアノ | 自動ピアノ演奏 | 不要 |
| 自動スクロール | ページの自動スクロール | 不要 |

## クイックスタート

1. [Releases](https://github.com/1bananachicken/MaaNTE/releases) から最新版をダウンロード
2. 解凍して実行ファイルを起動
3. タスクリストから必要な機能を選択
4. 開始して自動化をお楽しみください！

詳細なセットアップ手順は上流の [README](https://github.com/1bananachicken/MaaNTE) をご覧ください。

問題が発生した場合は、[トラブルシューティング](/ja_jp/トラブルシューティングマニュアル.html) と [FAQ](/ja_jp/develop/faq.html) をご確認ください。
